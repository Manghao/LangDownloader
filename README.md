# LangDownloader

> Download Dofus 1.29 langs file up-to-date

## Version
+ 1.0.1

## Author
+ Manghao

## Requirements
+ NodeJS v9+

## Setup
```shell
$ git clone git@github.com:Manghao/LangDownloader.git
$ cd LangDownloader
$ npm i
```

## Usage
```shell
$ node index.js
```

## Output
> Langs files are downloaded to **out** folder
